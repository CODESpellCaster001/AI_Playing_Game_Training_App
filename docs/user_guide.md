
应用程序主窗口用户指南
该应用程序的主窗口将出现，并通过不同的部分提供各种功能的访问入口。

模型训练
训练模型的步骤：
选择游戏环境：

在“训练模型”部分，从下拉菜单中选择所需的游戏环境。

设置训练参数：

在“时间步数”字段中输入训练的时间步数。

在“向量堆叠数”字段中输入向量堆叠的数量。

在“环境数量”字段中输入并行环境的数量。

选择目录：

点击“选择保存目录”按钮，选择训练模型的保存位置。

点击“选择日志目录”按钮，选择 TensorBoard 日志的保存位置。

开始训练模型：

点击“训练模型”按钮开始训练模型。训练进度会通过终端输出进行显示。

TensorBoard 可视化
查看训练结果的步骤：
选择日志目录：

在“查看训练结果”部分，点击“选择日志目录”按钮，选择包含 TensorBoard 日志的目录。

显示 TensorBoard：

点击“显示 TensorBoard”按钮启动 TensorBoard。终端将显示一个可以在浏览器中访问的地址。

智能体监控
监控智能体的步骤：
选择模型文件：

在“监控智能体”部分，点击“选择模型文件”按钮，选择已训练的模型文件。

设置监控参数：

从下拉菜单中选择游戏环境。

在“环境数量”字段中输入环境数。

在“向量堆叠数”字段中输入堆叠数。

在“录制时间（秒）”字段中输入录制时长。

开始监控：

点击“监控智能体”按钮开始监控智能体，性能将被录制为视频。

系统会提示选择保存视频的位置。


设置监控参数：

从下拉菜单中选择游戏环境。

在“环境数量”字段中输入环境数量。

在“录制时间（秒）”字段中输入录制时间。

故障排查
常见问题与解决方法：
未选择模型文件：

请确保在监控智能体之前已选择模型文件。

未选择日志目录：

请确保在运行 TensorBoard 之前已选择日志目录。
